Our supplementary materials contain two items:
1. 'supp.pdf': the PDF file describes more details of our method and provides additional results.
2. 'index.html' (with the 'video/' folder): the web page indexes the supplementary videos and provides an interactive visualization of the results.

Please keep 'index.html' and the 'video/' directory in the same folder so that the relative video paths resolve correctly.

Our videos are encoded with the FFmpeg encoder x264, and you can find the codec details here:
https://trac.ffmpeg.org/wiki/Encode/H.264
